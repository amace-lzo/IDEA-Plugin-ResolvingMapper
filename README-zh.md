# IDEA-Plugin-ResolvingMapper

这是一个IDEA插件，可以解析Mapper.xml文件中所有的表名与存储过程名称，同时支持单个文件和多个文件。你不仅可以在展示这些信息在你的IDEA中，你也可以将其导出到指定路径。如果您有问题或者建议，请和我们联系。

## 支持标签

我们目前支持如下标签：

`select`、`insert`、`update`、`delete`、`sql`、`if`
